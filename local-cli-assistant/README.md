# Local CLI Assistant

A lightweight local terminal assistant built on Ollama.

## Version 1 Goals

- Chat mode
- Code mode for Python and shell help
- Safe helper tools
- Local-first workflow

## Models

- Chat: `llama3.1:latest`
- Code: `qwen2.5-coder:latest`

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
